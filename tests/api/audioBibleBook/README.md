## URL

`​/v1​/audio-bibles​/{audioBibleId}​/books​/{bookId}`

## What is this for

Gets a single `Book` object for a given **audioBibleId** and **bookId**.

## Request params

```ts
{
  'include-chapters'?: boolean;
}
```
